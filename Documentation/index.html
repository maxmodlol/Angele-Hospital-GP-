FAQ

Why images are not same as preview?

images are showed in demo only used for preview purpose, you can use your own images

How To Change Logo?

just replace your logos in "images" folder

Note: you need to modify some styles according to your logo height

Which Stylesheet i can modify?

Goto "CSS" folder edit "styles.css" and modify your changes

Can i write my own Javascript code?

Yes, use "custom.js" file, located in "js" folder

Can i use bootstrape elements in this template?

Yes, you can but you need to give your own styles

Charts

in ProClinic template we used two types of charts.

  • Morris Charts
  • Flot Charts

Sample Morris Charts


    var lineMorris = new Morris.Line({
    element: 'lineMorris',
    resize: true,
    data: [
      { y: '2017 Q1', item1: 31 },
      { y: '2017 Q2', item1: 18 },
      { y: '2017 Q3', item1: 70 },
      { y: '2017 Q4', item1: 32 },
      { y: '2018 Q1', item1: 15 },
      { y: '2018 Q2', item1: 53 }
    ],
    xkey: 'y',
    ykeys: ['item1'],
    labels: ['Item 1'],
    gridLineColor: '#eef0f2',
    lineColors: ['#EB3349'],
    lineWidth: 2,
    hideHover: 'auto'
  });

for more detailed documentation go to Morris Charts and Flot Charts
